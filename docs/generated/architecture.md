# Architecture

> Generated by `npm run ingest`. Source of truth remains the referenced implementation.

## System shape

React 19 + TypeScript + Vite frontend. Supabase provides authentication, database, storage, and Edge Functions. React Router owns navigation; Zustand owns auth state; React Query owns server cache.

## Boundaries

- UI: `src/pages/` and `src/components/`
- State: `src/store/` and `src/hooks/`
- Integration: `src/api/` and `supabase/functions/`
- Persistence: `supabase/migrations/`
- Agent operations: Supabase/database MCP for local and staging schema work; Playwright MCP on demand for UI smoke tests
