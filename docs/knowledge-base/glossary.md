# Glossary

> Generated by `npm run ingest`. Source of truth remains the referenced implementation.

- **Client**: authenticated tenant represented by a Supabase `clients` row.
- **Workflow**: configured automation connected to a client.
- **Run**: one reported workflow execution, keyed by `event_id`.
- **Feature type**: product-facing automation category shown in the dashboard.
- **Edge Function**: server-side Supabase function used for trusted integrations.
