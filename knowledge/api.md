# API

> Generated by `npm run ingest`. Source of truth remains the referenced implementation.

## Browser clients

- `src/api/supabase/auth.ts` — `getSession`, `signInWithEmail`, `signOut`
- `src/api/supabase/client.ts` — `supabase`
- `src/api/supabase/storage.ts` — `storage`, `uploadFile`

## Edge functions

- `supabase/functions/ingest-run/index.ts`
- `supabase/functions/invite-client/index.ts`

## MCP boundary

Supabase/database MCP access is limited to local and staging environments. Playwright/browser MCP is used only for local or staging smoke tests. Filesystem and Git remain Codex-native capabilities.

## Security

Browser code uses public Supabase credentials from `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`. Webhook and admin secrets stay in Edge Function environment variables.
